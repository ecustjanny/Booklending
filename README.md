# Booklending
Booklending project：通过区块链以太坊技术实现的图书馆去中心化的图书借阅和点评服务。
流程说明:
	图书服务包含两方：借出者和借入者。
	借出者发布当前书可以借出状态，借入者点击该书请求借阅。
	借出者可以点评该书（1-5星）。
  线下交书（线下当面，放在线下书柜中，并将箱子号码及密码发给借阅者，或者快递等）
	借阅者拿到书籍后，确认数据完好，点击确认，交易完成；或者发现书本有破损，拒绝借阅。
	（借出者拿到拒绝的书籍向图书馆发起还书或者回收请求。图书馆完成后续处理。不在这里实现）
